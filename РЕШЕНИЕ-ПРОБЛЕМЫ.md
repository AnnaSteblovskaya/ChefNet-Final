# Решение проблемы с турецкой версией FAQ2

## Проблема

В файле `/src/locales/translations.ts` на строке 1856 (турецкая версия FAQ2) используется **неправильное экранирование** символов новой строки:

- **Текущее состояние:** `\\\\n\\\\n` (четыре обратных слэша)
- **Должно быть:** `\\n\\n` (два обратных слэша)

Из-за этого буллеты не отображаются на отдельных строках в турецкой версии.

## Почему не удалось исправить автоматически?

К сожалению, из-за ограничений инструментов редактирования (особенности экранирования спецсимволов в строковых литералах TypeScript) автоматическое исправление не сработало. 

## Решение - Ручное исправление

### Вариант 1: Простая замена (РЕКОМЕНДУЕТСЯ)

1. Откройте файл `/src/locales/translations.ts` в редакторе кода (VS Code, WebStorm и т.д.)

2. Перейдите к строке **1856** (используйте Ctrl+G или Cmd+G и введите "1856")

3. Вы увидите строку, которая начинается с:
   ```typescript
   "Hibrit bir Pazar Yeri + SaaS + Fintech modeliyle çalışıyoruz...
   ```

4. Найдите в этой строке ВСЕ вхождения `\\n\\n` (это выглядит как два обратных слэша, потом буква n, еще два обратных слэша, еще раз буква n)

5. Замените каждое `\\n\\n` на `\n\n` (один обратный слэш, буква n, один обратный слэш, буква n)

   **Внимание!** В этой строке таких вхождений **ЧЕТЫРЕ**.  Нужно заменить все 4.

6. Сохраните файл (Ctrl+S или Cmd+S)

### Вариант 2: Поиск и замена через VS Code

1. Откройте `/src/locales/translations.ts`

2. Нажмите **Ctrl+H** (или **Cmd+H** на Mac) для открытия панели "Поиск и замена"

3. В поле "Find" (Найти) введите:
   ```
   çeşitlendiriyoruz:\\n\\n
   ```

4. В поле "Replace" (Заменить) введите:
   ```
   çeşitlendiriyoruz:\n\n
   ```

5. Нажмите "Replace All" или замените каждое вхождение вручную (их 4 в одной строке 1856)

6. Сохраните файл

### Проверка

После исправления строка 1856 должна быть такой же как строка 1853 (faq1Answer) по формату экранирования:

**Правильно (как в строке 1853):**
```typescript
"...текст...\\n\\n..."  // ДВА обратных слэша
```

**Было неправильно (строка 1856):**
```typescript
"...текст...\\\\n\\\\n..."  // ЧЕТЫРЕ обратных слэша
```

## Проверка результата

После исправления перезагрузите страницу и переключитесь на турецкий язык. Во втором вопросе FAQ должны появиться буллеты на отдельных строках, как в русской, английской и немецкой версиях.

---

**P.S.** Эта проблема возникла из-за того, что при копировании текста из какого-то источника символы экранирования были продублированы. В будущем при редактировании переводов следите за правильным форматом: `\\n\\n` а не `\\\\n\\\\n`.
