# Автоматическое исправление турецкого FAQ2

## Проблема
В турецкой версии FAQ2 (строка 1856 файла `/src/locales/translations.ts`) используется неправильное экранирование символов новой строки, из-за чего буллеты не отображаются на отдельных строках.

## Решение

### Вариант 1: Через текстовый редактор (рекомендуется)

1. Откройте файл `/src/locales/translations.ts`
2. Перейдите к строке 1856 (Ctrl+G или Cmd+G)
3. Найдите и замените во ВСЕЙ СТРОКЕ 1856:

**Было (4 обратных слэша):**
```
\\n\\n
```

**Должно быть (2 обратных слэша):**
```
\n\n
```

Это нужно сделать для ВСЕХ вхождений `\\n\\n` в этой строке (их там 4 штуки).

### Вариант 2: Поиск и замена в VS Code

1. Откройте `/src/locales/translations.ts`
2. Нажмите Ctrl+H (или Cmd+H на Mac)
3. Включите режим регулярных выражений (кнопка `.*`)
4. В поле "Find" введите:
   ```
   (faq2Answer:\s*"[^"]*?)\\\\\\\\n
   ```
5. В поле "Replace" введите:
   ```
   $1\\n
   ```
6. Нажмите "Replace All"

### Результат

После исправления строка 1856 должна выглядеть так (с переносами для наглядности):

```typescript
      "Hibrit bir Pazar Yeri + SaaS + Fintech modeliyle çalışıyoruz; 
      gelir akışlarımızı dört farklı kanaldan çeşitlendiriyoruz:\n\n
      • İşlem Ücretleri: Rezervasyonlar, yemek ön siparişleri ve etkinlik bilet satışlarından alınan yüzde payı.\n\n
      • Abonelikler (B2B SaaS): Restaurantların CRM sistemlerine, yapay zekâ destekli analizlere ve pazarlama araçlarına erişim karşılığında ödediği aylık ücretler.\n\n
      • Fintech (Ödemeler): Uygulama içi ödemelerden ve hesap bölme özelliklerinden elde edilen işlem ücretleri.\n\n
      • Reklam: Uygulama içindeki keşfetme ve arama sonuçlarında restaurantların ücretli tanıtımı.",
```

Обратите внимание: `\n\n` (а не `\\n\\n` как было)
