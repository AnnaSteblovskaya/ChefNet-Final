# 📋 ИНСТРУКЦИЯ ПО ИСПРАВЛЕНИЮ

## Что нужно сделать

Запустить скрипт, который добавит буллеты (•) в турецкую версию второго вопроса FAQ, чтобы форматирование соответствовало русской, английской и немецкой версиям.

## Как запустить

### Способ 1: Через терминал (РЕКОМЕНДУЕТСЯ)

1. Откройте терминал в корневой папке проекта
2. Скопируйте и вставьте эту команду:

```bash
node RUN-THIS.js
```

3. Нажмите Enter
4. Готово! Вы увидите сообщение об успешном выполнении

### Способ 2: Если первая команда не работает

Попробуйте полный путь:

```bash
node /RUN-THIS.js
```

### Способ 3: Через npm/npx

```bash
npx node RUN-THIS.js
```

## Что делает скрипт?

Скрипт автоматически:
1. Читает файл `/src/locales/translations.ts`
2. Находит турецкую версию `faq2Answer`
3. Добавляет символ `•` перед каждым из 4 пунктов:
   - • İşlem Ücretleri:
   - • Abonelikler (B2B SaaS):
   - • Fintech (Ödemeler):
   - • Reklam:
4. Сохраняет изменения

## Проверка результата

После запуска скрипта:
1. Откройте сайт
2. Переключитесь на турецкий язык (🇹🇷)
3. Перейдите в раздел FAQ
4. Откройте второй вопрос "ChefNet'in iş modeli nasıl işliyor?"
5. Убедитесь, что все 4 пункта отображаются с буллетами

## Альтернативные скрипты

Если по какой-то причине `RUN-THIS.js` не работает, попробуйте:

- `node do-fix.js`
- `node SIMPLE-FIX.js`
- `node fix-line-1549.js`
- `node FINAL-FIX.js`

Все эти скрипты делают одно и то же, но с разным подходом.
