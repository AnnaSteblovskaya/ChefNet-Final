=== ИНСТРУКЦИЯ ПО ПРИМЕНЕНИЮ ===

1. Откройте файл /src/locales/translations.ts
2. Найдите соответствующий раздел для каждого языка
3. Замените строки с seedBackTitle, seedBackDesc, privateBackTitle, privateBackDesc, 
   marketingBackTitle, marketingBackDesc, publicBackTitle, publicBackDesc
4. Сохраните файл

ВАЖНО: 
- Все bullet points используют символ • (не дефис -)
- Все переносы строк используют \\n (с двумя обратными слэшами)
- Заголовки теперь содержат описание вместо сумм (Seed: $150,000 → Разработка MVP приложения)
