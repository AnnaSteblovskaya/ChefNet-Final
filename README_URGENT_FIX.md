## 📋 Описание проблемы

Файл `/src/locales/translations.ts` содержит **критическую ошибку**:
- **Строки 1101-1171:** Дублированная поврежденная секция `es: {}` с мусорными данными
- **Причина поломки:** Смешанный немецкий/английский текст, множественные экранированные слеши
- **Последствие:** Полная остановка компиляции приложения

```bash
node fix-translations.js
```

- Читает файл translations.ts
- Удаляет поврежденные строки 1101-1171
- Создает резервную копию `.backup`

**Время выполнения:** ~1 секунда

### 🔧 Способ 4: Ручное исправление через редактор

**В VS Code / WebStorm:**
1. Откройте `/src/locales/translations.ts`
2. Нажмите `Ctrl+G` (Windows/Linux) или `Cmd+G` (Mac)
3. Введите `1101` и нажмите Enter
4. Выделите строки от 1101 до 1171 включительно
5. Нажмите Delete
6. Сохраните файл (`Ctrl+S` / `Cmd+S`)

## 🆘 Если ничего не помогает

Если ни один из автоматических скриптов не работает:

1. **Создайте резервную копию вручную:**
   ```bash

   ```

2. **Выполните через командную строку:**
   ```bash
   # Linux/Mac
   head -n 1100 src/locales/translations.ts > temp.ts
   tail -n +1172 src/locales/translations.ts >> temp.ts
   mv temp.ts src/locales/translations.ts
   
   # Windows PowerShell
   $content = Get-Content "src\locales\translations.ts"
   $fixed = $content[0..1099] + $content[1171..($content.Length-1)]
   $fixed | Set-Content "src\locales\translations.ts"
   ```

3. **Или откройте файл в редакторе и удалите строки 1101-1171 вручную**
