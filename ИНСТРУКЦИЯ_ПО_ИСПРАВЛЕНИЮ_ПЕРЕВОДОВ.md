# Инструкция по исправлению файла translations.ts

## Проблема
В файле `/src/locales/translations.ts` обнаружены дублирующиеся секции языков:

### Структура ошибок:
1. **Строки 1-1160**: ✅ Правильные секции (en, ru, de, es)
2. **Строки 1161-1379**: ❌ НЕПРАВИЛЬНАЯ турецкая секция (tr) - содержит французский текст
3. **Строки 1380-1610**: ❌ ДУБЛИРУЮЩАЯСЯ испанская секция (es)
4. **Строки 1611-1841**: ✅ ПРАВИЛЬНАЯ турецкая секция (tr)

### Подтверждение ошибки:
- Строка 1230 содержит французский текст: `feature3Title: 'Flexibilité d\'utilisation'` (должен быть турецкий)
- Турецкая секция встречается дважды: строка 1161 и строка 1611
- Испанская секция встречается дважды: строка 930 и строка 1380

## Решение

### Вариант 1: Автоматическое исправление (рекомендуется)
Выполните следующие команды в терминале:

```bash
# Сохраните резервную копию
cp src/locales/translations.ts src/locales/translations.ts.backup

# Создайте исправленный файл (удаляем строки 1161-1610)
head -n 1160 src/locales/translations.ts > /tmp/part1.txt
tail -n +1611 src/locales/translations.ts > /tmp/part2.txt
cat /tmp/part1.txt /tmp/part2.txt > src/locales/translations.ts

# Проверьте количество строк (должно быть 1395)
wc -l src/locales/translations.ts
```

### Вариант 2: Ручное исправление
1. Откройте файл `/src/locales/translations.ts` в редакторе
2. Перейдите к строке 1160
3. Удалите все строки с 1161 по 1610 включительно (всего 450 строк)
4. Сохраните файл

### Проверка правильности:
После исправления файл должен иметь следующую структуру:

```
import { Language } from '@/contexts/LanguageContext';

type TranslationKeys = { ... }

export const translations: Record<Language, TranslationKeys> = {
  en: { ... },    // Английский
  ru: { ... },    // Русский  
  de: { ... },    // Немецкий
  es: { ... },    // Испанский (ОДНА секция)
  tr: { ... },    // Турецкий (ОДНА секция, без французского текста)
};

export const getTranslation = ...
```

## Результат

- ✅ 5 языков: English, Русский, Deutsch, Español, Türkçe
- ✅ Нет дублирующихся секций
- ✅ Турецкая секция содержит только турецкий текст
- ✅ Файл содержит ~1395 строк вместо 1845
