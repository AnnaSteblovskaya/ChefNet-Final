ChefNet Invest - автоматическое исправление

⚠️ Внимание! Эта страница содержит инструкции для исправления проблемы с экранированием символов в турецкой версии FAQ2.

📌 Проблема

В файле /src/locales/translations.ts на строке 1856 используется неправильное экранирование:

Текущее: \\\\n\\\\n (4 обратных слэша)

Должно быть: \\n\\n (2 обратных слэша)

✅ Решение через VS Code (РЕКОМЕНДУЕТСЯ)

Шаг 1: Откройте файл /src/locales/translations.ts

Шаг 2: Нажмите Ctrl+H (или Cmd+H на Mac)

Шаг 3: Включите режим регулярных выражений (кнопка .*)

Шаг 4: В поле "Find" введите:

\\\\\\\\n\\\\\\\\n

Шаг 5: В поле "Replace" введите:

\\\\n\\\\n

Шаг 6: Нажмите "Replace All" (должно заменить 4 вхождения)

Шаг 7: Сохраните файл (Ctrl+S или Cmd+S)

🖱️ Альтернатива: Ручное редактирование

1. Перейдите к строке 1856 (Ctrl+G → введите "1856")

2. Найдите в этой строке все \\n\\n

3. У каждого \\n\\n удалите ОДИН обратный слэш перед \n

4. Повторите для всех 4 вхождений в строке

✅ Проверка

После исправления строка 1856 должна иметь такое же экранирование, как строка 1853 (faq1Answer):

"...текст...\\n\\n..." (ДВА обратных слэша, не четыре)

🎯 Результат

После сохранения файла перезагрузите страницу в браузере. В турецкой версии FAQ второй вопрос должен отображаться с буллетами на отдельных строках, как в русской, английской и немецкой версиях.
