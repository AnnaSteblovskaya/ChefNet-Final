# 🚨 КРИТИЧЕСКОЕ ИСПРАВЛЕНИЕ FAQ1 (Испанский)

## Проблема
В файле `/src/locales/translations.ts` на строке **1516** всё ещё остаются **ЧЕТЫРЕ** обратных слеша `\\\\n\\\\n` вместо **ДВУХ** `\\n\\n`.

## Текущее состояние (НЕПРАВИЛЬНО):
```
privados.\\\\n\\\\n Aborda
```

## Должно быть (ПРАВИЛЬНО):
```
privados.\\n\\nAborda
```

---

## Метод 1: Поиск и замена в редакторе

### Если у вас VS Code:
1. Откройте `/src/locales/translations.ts`
2. Нажмите `Ctrl+F` (или `Cmd+F` на Mac)
3. В поле поиска введите точно (скопируйте):
   ```
   privados.\\\\n\\\\n Aborda
   ```
4. В поле замены введите точно (скопируйте):
   ```
   privados.\\n\\nAborda
   ```
5. Нажмите "Replace" (Заменить)
6. Сохраните файл (`Ctrl+S` или `Cmd+S`)

---

## Метод 2: Замена всей строки 1516

Замените **всю строку 1516** на:
```typescript
      "ChefNet es una plataforma global de FoodTech que unifica el descubrimiento, reservas, pagos y engagement con restaurantes y chefs privados.\\n\\nAborda la fragmentación del ecosistema de servicios de alimentación mediante el establecimiento de un único estándar integrado y sin fricciones.",
```

**ВАЖНО:** 
- В этой строке должно быть ровно **ДВА** обратных слеша перед каждой буквой `n`
- Не должно быть пробела после `\\n\\n`
- Строка начинается с 6 пробелов (отступ)

---

## Проверка после исправления

После исправления:
1. Сохраните файл
2. Обновите страницу в браузере
3. Откройте FAQ1 на испанском языке
4. Должен быть виден перенос строки между двумя предложениями

---

## Почему это не работает автоматически?

Некоторые текстовые редакторы автоматически экранируют обратные слеши при сохранении, превращая `\\n` в `\\\\n`. Нужно следить, чтобы после сохранения в файле осталось именно `\\n\\n` (два обратных слеша + n + два обратных слеша + n).
